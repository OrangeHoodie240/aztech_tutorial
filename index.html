Changing Your Own Password


Find and click the change password button in the top right corner of the screen.

After you click on it, the button will change to say "Hide Form", which when clicked will hide the form from view.




Fill out the form and submit.



Changing A Customers Password


Locating the change customer password section, enter the client's username and click the buttom.

You will soon receive a randomized password in green text. This is a temp password to be given to the user. The user should be instructed to change this randomized password once they log in.



Creating A User


Find the create user button in the top right of the screen.

After clicking the button text will change to "Hide Form", which when clicked again, will hide the form.




Fill out the information for the user

Be sure to check the admin check box if the user is to be an admin.




After successful user creation, a temporary password will be displayed in a green font. This is a randomally generated password that is to be given to the user. They should be instructed to change this password when they log in.

Inspect User By Account

This method can only be used to inspect users that have an account. If a user does not have an account, can always inspect by their customer number



Because the list of users can be quite large, we can use the filter bar to hone in on the user account we are looking for.



Once we have located the user account we want to inspect, click the user row. The row will turn orange showing that the user has been selected.

Finally click the Inspect User button.

Inspect User By Customer Number

We can inspect a user by their customer number whether or not they currently have a portal account.

Simply type the customer number in the input box and either press enter or click inspect user.

Navigating An Inspected User

The customer information is divided into four tables. We select a table in order to view it. The currently selected table is lighter shade of orange than the others.

We have search boxes above the user table that allow us to filter the rows in the selected table by the values of certain columns.

We can click the columns in order to reorder the rows by their value in the clicked column.

Finding A User's Certificates

The certificates associated with an order are to be found on the user's shipment history page.

On this table there will be a column named Certificates?. This will be have either a Yes or No value. Those rows with a Yes value will have certificates attached. We simply click the row to select it and a list of links to all the certificates associated to that row will pop up.

Exiting User Inspection

To exit inspection of a user and to return to the admin page, simply click the button with the text "Clear Inspection".
